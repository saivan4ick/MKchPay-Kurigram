# MKchPay-Kurigram
Бот для автоматической продажи цифровых товаров (пасскодов) через встроённую платёжную систему Telegram Stars. После оплаты покупатель мгновенно получает уникальный пасскод в личном сообщении.

Реализован на основе [Pyrogram](https://pyrogram.org/) / [Kurigram](https://github.com/KurimuzonAkuma/pyrogram).

---

## Возможности

- 🛒 Продажа пасскодов за Telegram Stars (валюта `XTR`)
- ⚡ Мгновенная автоматическая выдача пасскода после успешной оплаты
- 📦 Отображение актуального остатка товара в реальном времени
- 🛡 Защита от дублирования оплат по `charge_id`
- 🛠 Панель администратора с управлением через кнопки
- 👥 Поддержка нескольких администраторов
- 📂 Загрузка пасскодов через `.txt`-файл с автоматической фильтрацией дублей

---

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главная страница магазина с кнопкой покупки |
| `/buy` | Сразу открыть форму оплаты |
| `/admin` | Панель администратора (только для админов) |
| `/cancel` | Отменить текущее действие (загрузку, добавление админа) |

## Требования

- Python **3.11+** (рекомендуется 3.12+)
- Аккаунт на [my.telegram.org](https://my.telegram.org) для получения `API_ID` и `API_HASH`
- Telegram-бот, созданный через [@BotFather](https://t.me/BotFather)
- Подключённые Telegram Payments (Stars) в настройках бота у BotFather

## Установка

### 1. Клонировать репозиторий

```bash
git clone https://github.com/ВАШ_НИК/MKchPay-Kurigram.git
cd MKchPay-Kurigram
```

### 2. Установить зависимости

**Через pip:**
```bash
pip install -r requirements.txt
```

**Через uv (рекомендуется):**
```bash
uv sync
```

### 3. Настроить переменные окружения

Скопируйте пример конфига и заполните его:

```bash
cp .env.example .env
```

Откройте `.env` и укажите значения (подробнее в разделе [Конфигурация](#конфигурация)).

### 4. Создать файл с пасскодами

Создайте файл `passcodes.txt` (или другой, указанный в `PASSCODES_FILE`) и добавьте пасскоды — по одному на каждой строке:

```
PASSCODE-AAAA-1111
PASSCODE-BBBB-2222
PASSCODE-CCCC-3333
```

### 5. Запустить бота

```bash
python bot.py
```

## Конфигурация

Все настройки задаются через файл `.env` в корне проекта.

| Переменная | Обязательная | По умолчанию | Описание |
|---|---|---|---|
| `BOT_TOKEN` | ✅ | — | Токен бота от [@BotFather](https://t.me/BotFather) |
| `API_ID` | ✅ | — | API ID приложения с [my.telegram.org](https://my.telegram.org) |
| `API_HASH` | ✅ | — | API Hash приложения с [my.telegram.org](https://my.telegram.org) |
| `OWNER_ID` | ✅ | — | Telegram ID владельца бота (получить через [@userinfobot](https://t.me/userinfobot)) |
| `PRICE_STARS` | ❌ | `1` | Цена пасскода в Telegram Stars |
| `PASSCODES_FILE` | ❌ | `passcodes.txt` | Путь к файлу с пасскодами |
| `ADMINS_FILE` | ❌ | `admins.txt` | Путь к файлу со списком дополнительных администраторов |

### Пример `.env`

```env
BOT_TOKEN=1234567890:ABCdefGHijklMNOpqrSTUvwxYZ
API_ID=12345678
API_HASH=abcdef1234567890abcdef1234567890
OWNER_ID=987654321

PRICE_STARS=50
PASSCODES_FILE=passcodes.txt
ADMINS_FILE=admins.txt
```

### Как получить API_ID и API_HASH

1. Зайдите на [my.telegram.org](https://my.telegram.org) и войдите через номер телефона
2. Перейдите в раздел **API development tools**
3. Создайте приложение, заполнив поля (название и платформа — любые)
4. Скопируйте `App api_id` и `App api_hash`

### Включение Telegram Stars у BotFather

1. Откройте [@BotFather](https://t.me/BotFather) и выберите вашего бота
2. Перейдите в **Bot Settings → Payments**
3. Включите **Telegram Stars** как способ оплаты

## Управление пасскодами

### Через файл вручную

Добавьте пасскоды в файл `passcodes.txt` (по одному на строку). Бот выдаёт их по порядку — сверху вниз.

### Через панель администратора

1. Отправьте `/admin` боту
2. Нажмите **📂 Загрузить пасскоды из файла**
3. Отправьте `.txt`-файл с пасскодами

Дубликаты автоматически пропускаются при загрузке.

---

## Панель администратора

Доступна через команду `/admin`. Открывается только для владельца (`OWNER_ID`) и добавленных администраторов.

**Возможности:**
- 📦 **Статистика** — текущий остаток пасскодов и число администраторов
- ➕ **Добавить администратора** — добавить нового админа по Telegram ID
- 📂 **Загрузить пасскоды из файла** — загрузить новую партию пасскодов

> Владелец (`OWNER_ID`) всегда имеет доступ к панели и не может быть удалён.
